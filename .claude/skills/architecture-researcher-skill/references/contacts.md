# Architecture Contacts Directory

Add known Design Authorities, Domain Architects, and key Application Owners here.
The research agent checks this file before looking them up in registries.

## Format

```
## [Domain Name]

| Role | Name | Contact | Scope | Notes |
|------|------|---------|-------|-------|
| Design Authority | [Name] | [email/Slack] | [what they govern] | [availability, preferences] |
| Domain Architect | [Name] | [email/Slack] | [what they own] | |
| App Owner | [Name] | [email/Slack] | [application name] | |
```
